enchant.oes.OesChapter(scenario , builder)
↑
oes章のクラス。enchant.Group を継承。 直接このクラスのインスタンスを生成する必要はない。 章を生成してシナリオに追加するには enchant.oes.OesScenario.addChapter() を使う。 addLine() やそれを内包したメソッドを使用して章を構築する。
引数

Function(enchant.oes.OesChapter) builder 章を構築する関数。

Function print(oesText , text)
↑
逐次文字出力。（これは addLine() 内包メソッド）。 テキストにhtmlタグや独自タグを埋め込める。 ただし、htmlタグは　a,br,span　など一部のタグの動作しか確認していないので注意。
独自タグは以下のものが使用できる。
{kw} 入力待ち。
{kl} 入力待ち後改行。
{kp} 入力待ち後改ページ。
{np} 改ページ。
{wt=nnn} 逐次出力の待ち時間変更。nnn=秒。
{wt} 逐次出力の待ち時間をデフォルトに戻す。
{wd=nnn} 逐次出力の待ち時間変更。nnn=デフォルト待ち時間に対する倍率。
{wd} 逐次出力の待ち時間をデフォルトに戻す。
※完了後は次の行へ進む。
引数
enchant.oes.oesText oesText OesTextオブジェクト。
String text テキスト。
戻り値
Function 行に登録された関数オブジェクト。
サンプル 1
  1 chapter.print(txt,'入力待ち{kw}入力待ち後改行{kl}入力待ち後改ページ{kp}');
≫プレインテキスト
chapter.print(txt,'入力待ち{kw}入力待ち後改行{kl}入力待ち後改ページ{kp}');
≫ハイライト
サンプル 2

≫プレインテキスト

Function tween(targets [, waitobj])
↑
対象を指定してtweenを実行する。（これは addLine() 内包メソッド）。 複数対象の場合はその中の最大所要時間がこのtweenの所要時間となる。 waitobj を指定した場合は非同期tweenとなる。
※完了後は次の行へ進む。
引数
Object | Object[ ] targets 対象のプロパティ。
Object [waitobj] 非同期tween時の待ち合わせ用オブジェクト。
戻り値
Function 行に登録された関数オブジェクト。
サンプル 1
  1 // fade in
  2 chapter.act(function() {
  3 	text.opacity = 0;
  4 )};
  5 chapter.tween(
  6 	{node:text, time:game.fps, opacity:1, easing:enchant.Easing.QUAD_EASEIN}
  7 );
