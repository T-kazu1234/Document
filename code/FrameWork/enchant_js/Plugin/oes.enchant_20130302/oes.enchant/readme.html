oes.enchant.jsとは？

oes.enchant.js はノベルゲーム向けの機能を提供する enchant.js のプラグインです。
enchant.js は HTML5 + JavaScript なゲームエンジンです。

注意

enchant.js v0.6 では描画エンジンが変更されるため、本プラグインは動作しません！
v0.5以下を使用してください。
v0.6についてはこちらを参照してください。

同梱ファイルおよびフォルダ

enchant.js, enchant.min.js, tl.enchant.js
本プラグインの動作を確認するのに使用した enchant.js および関連ファイル。
ex.enchant.js, ex.enchant.min.js
enchant.jsのクラスをprototype拡張した追加機能のプラグイン。
ex.enchant.min.js は ex.enchant.js の内容を短く縮めたファイル。
oes.enchant.js, oes.enchant.min.js
ノベルゲーム向けの機能を提供するプラグイン。
oes.enchant.min.js は oes.enchant.js の内容を短く縮めたファイル。
oesUtil.enchant.js
セーブ／ロード画面やログ画面などのユーティリティ機能を提供するプラグイン。
doc/
ex.enchant.js 及び oes.enchant.js のAPIリファレンス。

使用方法

htmlの<script>タグで組み込んでください。
例
<script type="text/javascript" src="enchant.min.js"></script>
<script type="text/javascript" src="tl.enchant.js"></script>
<script type="text/javascript" src="ex.enchant.min.js"></script>
<script type="text/javascript" src="oes.enchant.min.js"></script>
<script type="text/javascript" src="oesUtil.enchant.js"></script>
